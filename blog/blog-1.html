Inline vs Inline Blocks

30/06/2021

Inline and Inline Block are two ways to specify the display of an element. They determine how the element is going to be placed in the structure of the webpage. They also determine how you can affect the position of that element or the dimensions of it e.g the width or height of the element.

Inline

Inline effectively displays each element in one line until the width of the content is too much to fit on the page, where it then starts a new line and places elements on that line. This means that you cannot control the height or width of each element, and also that the position of each element is simply in the order that they are placed on the page.

Block

Block on the other hand starts on a new line with each element, and each element will take up the full width of the screen. You can set your own width and height values for Block which is helpful for images and other shapes that you may want to edit on your page.

Inline-Block

Inline-Block is a combination of Inline and Block because it allows the user to change the width and height of the element as in Block, but also allows elements to sit next to each other (they do not have to each be on a new line) similar to the way that Inline works with placing elements next to each other.
